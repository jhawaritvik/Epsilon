Design Principles

Core architectural decisions for scientific rigor

üîí Epistemic Integrity

Evaluation Agent is isolated from Design Agent to prevent p-hacking and preserve scientific rigor.

üîÑ Self-Correction

Feedback loop with up to N iterations allows the system to learn from failures and refine experiments.

üìÅ Run Isolation

Each run gets its own experiments/{run_id}/ directory preventing race conditions and ensuring reproducibility.

üíæ Memory Persistence

All findings stored in Supabase enable cross-run learning and knowledge accumulation.

üìù Auditability

Complete audit trail in Run Memory allows full traceability of every decision and iteration.

üî¨ Reproducibility

All code, data artifacts, and configurations are preserved for experiment reproducibility.
